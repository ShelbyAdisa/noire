Parfum ARMANI CODE

Strong yet sensitive, Armani Code Parfum reinvents the olfactory signature of the original Armani Code fragrance.

Retaining its woody aromatic and sensual identity, this fragrance captures the essence of Armani Code.

It opens with aromatic top notes that exude freshness, balanced by powerfully masculine base notes. The heart of the fragrance is now infused with an amplified sensitivity, creating a scent that is both intense and sophisticated.

Armani Code Parfum embodies a new, powerful yet sensitive olfactory signature, crafted for the modern man who appreciates timeless elegance with a bold edge.

"A reinvention of strength and sensitivity." - Noire Luxe x Armani
